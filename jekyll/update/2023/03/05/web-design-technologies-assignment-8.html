Introduction

In this module we learned how we can access data from forms on an HTML document in PHP. There were two methods of doing this described, GETs and POSTs

GET Requests

The difference between a GET and POST request is that the GET has the payload in the URL bar. For example, when you search something on Google the parameter is “q=”, so this is a valid query: https://google.com/search?q=example

And this is what Google’s server would actually end up receiving from your browser:

GET /search?q=example HTTP/1.1
Host: google.com

In PHP, you can access the parameters from the $_GET object. You do this by indexing it with a key. For example, the Google query would look like this as a data dictionary:

{
   "q": "example"
}

“q” would be the key, so Google would access your query like this if they were using PHP:

$query = $_GET["q"]

POST Requests

With POST requests, a payload body is sent. Instead of having the parameters sent values are sent in the payload body. For example, if the Google search handled POSTs it would receive something like this:

POST /search HTTP/1.1
Host: google.com

q=example

Which Google could then access similar to the GET request, but using the $_POST variable.

$query = $_POST[“q”]

You can use forms in HTML to send data to an endpoint (such as /search). By default the browser will send a GET request but it can also do POSTs with the “method=” attribute.

Here’s an example form with POST:

<form action="/action.php" method="post">
   <label for="fname">First name:</label>
   <input type="text" id="fname" name="fname"><br><br>
   <label for="lname">Last name:</label>
   <input type="text" id="lname" name="lname"><br><br>
   <input type="submit" value="Submit">
</form>

And then the accompanying PHP code in action.php would look like this:

<?php

$firstName = $_POST["fname"]
$lastName = $_POST["lname"]

?>
<html>
<body>
<h1>Hello <?php echo $firstName; ?>. Your last name is <?php echo $lastName; ?>
</body>
</html>

It would be similar for GETs, just change the method=”post” to method=”get” on the form, and the $_POST to $_GET on action.php.

Assignment

For this assignment I created a contact form in which a user can leave their name, email, and a message. The form uses a POST request to contact.php to echo back the message in a formatted string.

HTML Form:

<form method="post" action="contact.php">
		<label>First Name:</label><br>
		<input type="text" name="first_name"><br><br>
		
		<label>Last Name:</label><br>
		<input type="text" name="last_name"><br><br>
		
		<label>Email:</label><br>
		<input type="email" name="email"><br><br>
		
		<label>Message:</label><br>
		<textarea name="message"></textarea><br><br>
		
		<input type="submit" value="Submit">
	</form>

PHP Code (contact.php):

<?php
// Check if all required fields are present
if (empty($_POST['first_name']) || empty($_POST['last_name']) || empty($_POST['email']) || empty($_POST['message'])) {
    // Redirect back to the contact form with an error message
    header("Location: index.php");
    exit();
}

// Sanitize the input data
$first_name = htmlspecialchars($_POST['first_name']);
$last_name = htmlspecialchars($_POST['last_name']);
$email = htmlspecialchars($_POST['email']);
$message = htmlspecialchars($_POST['message']);
?>

<!DOCTYPE html>
<html>
<head>
	<title>Contact Form Submission</title>
</head>
<body>
	<h1>Contact Form Submission</h1>
	<p>Thank you for submitting the following information:</p><br />
	<strong>First Name:</strong> <?php echo $first_name; ?><br />
	<strong>Last Name:</strong> <?php echo $last_name; ?><br />
	<strong>Email:</strong> <?php echo $email; ?><br />
	<strong>Message:</strong> <?php echo $message; ?><br />
	</ul>
</body>
</html>
