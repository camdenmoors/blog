Introduction

In this module we covered the basics of connecting a PHP website to a MySQL database using the built-in connectors of PHP. We also talked about what a relational database is, why it’s important, and when to use them.

Relational Databases

Relational Databases have three layers:

  • Entity (Thing/Object)
  • Atrributes (Descriptions of the entity)
  • Entity instance

Every table (entity list) contains columns (attributes) and rows (instances of the entity). The benefit of a relational databases is how it is possible to create links between different entity lists by the attributes. For example, if you have multiple books written by a single author, you could have two tables, a books table and an authors table.

Authors Table:

ID Name Birthday Genres
1 Sarah 1/1/1970 fiction,nonfiction,scientific
2 Alex 1/1/1971 narrative,novel,science fiction
3 David 1/1/1972 mystery

Books Table:

ID Name Rating Authors
1 The Girl Who Vanished into Thin Air 4.5 1
2 Science of Happiness: A Guide to a Fulfilling Life 2.7 2
3 The Last Survivors of the Zombie Apocalypse 5 4
4 The Curse of the Phantom Thief 3.2 1,3

Where authors can have multiple books and books can have multiple authors (a many-to-many relationship).

Data Types

There are a large variety of data types in MySQL, such as:

Type Description
INT A whole number without decimals
BIGINT A large whole number without decimals
FLOAT A number with a decimal point
DOUBLE A large number with a decimal point
DECIMAL A precise number with decimal point
DATE A date in YYYY-MM-DD format
DATETIME A date and time in YYYY-MM-DD HH:MM:SS format
TIMESTAMP A timestamp representing a date and time
TIME A time in HH:MM:SS format
YEAR A year in four-digit format (e.g. 2023)
CHAR A fixed-length character string
VARCHAR A variable-length character string
BLOB A binary large object, for storing large files
TEXT A variable-length text string
ENUM A list of predefined values

Assignment

For this assignment I had a contact form that lets the user send me a message. The message gets stored in a database and can be read at a later date.

Form:

<!DOCTYPE html>
<html>
<head>
	<title>Module 9 - Contact Form</title>
</head>
<body>
	<h1>Contact Me</h1>
	<form method="post" action="contact.php">
		<label>First Name:</label><br>
		<input type="text" name="first_name"><br><br>
		
		<label>Last Name:</label><br>
		<input type="text" name="last_name"><br><br>
		
		<label>Email:</label><br>
		<input type="email" name="email"><br><br>
		
		<label>Message:</label><br>
		<textarea name="message"></textarea><br><br>
		
		<input type="submit" value="Submit">
	</form>
</body>
</html>

contact.php (Recieves data from the form):

<?php
// Database configuration
$host = "sql207.epizy.com";
$username = "";
$password = "";
$dbname = "";

// Create connection
$conn = mysqli_connect($host, $username, $password, $dbname);
if (!$conn) {
    die("Database connection failed miserably: " . mysql_error());
}

// Get form values
$first_name = $_POST['first_name'];
$last_name = $_POST['last_name'];
$email = $_POST['email'];
$message = $_POST['message'];

// Insert message into database
$stmt = mysqli_prepare($conn, "INSERT INTO messages (first_name, last_name, email, message) VALUES (?, ?, ?, ?)");
mysqli_stmt_bind_param($stmt, "ssss", $first_name, $last_name, $email, $message);

if (mysqli_stmt_execute($stmt)) {
    echo "Message saved successfully";
} else {
    echo "Error: " . mysqli_stmt_error($stmt);
}

mysqli_stmt_close($stmt);

?>

messages.php (Lets you view the messages in the database)

<?php
// Database configuration
$host = "sql207.epizy.com";
$username = "";
$password = "";
$dbname = "";

// Create connection
$conn = mysqli_connect($host, $username, $password, $dbname);

if (!$conn) {
    die("Database connection failed miserably: " . mysqli_connect_error());
}

// Select all messages from the database
$sql = "SELECT * FROM messages";
$result = mysqli_query($conn, $sql);

if (mysqli_num_rows($result) > 0) {
    // Output data of each row
    while($row = mysqli_fetch_assoc($result)) {
        echo "Name: " . $row["first_name"] . " " . $row["last_name"] . "<br>";
        echo "Email: " . $row["email"] . "<br>";
        echo "Message: " . $row["message"] . "<br><br>";
    }
} else {
    echo "No messages found in the database";
}

// Close the connection
mysqli_close($conn);
?>
