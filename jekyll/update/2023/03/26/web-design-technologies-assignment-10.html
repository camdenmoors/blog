Introduction

In this module we covered the basics of creating a daabase with PHPMyAdmin, connecting to the MySQL database with PHP, and inserting rows with PHP.

Creating tables in PHPMyAdmin

When opening PHPMyAdmin there is a button on the left to create a table. Upon selecting you are presented with a list of columns to which you can add various data types. At minimum, there should be an ID to keep track of the items of the database.

As per the example in the lesson, a table of students could look like:

Name Type Length
ID int 4
name VARCHAR 50
lastName VARCHAR 50

Connection to MySQL

To connect PHP to your MySQL table, you can use either the MySQLi extension or PDO (PHP Data Objects). Both methods have their advantages, I will use MySQLi, as it is specifically designed for MySQL databases.

Here is a simple example of connecting to a MySQL table using PHP:

<?php
$servername = "localhost";
$username = "username";
$password = "password";

$conn = new mysqli($servername, $username, $password);

if ($conn->connect_error) {
  die("Connection failed: " . $conn->connect_error);
}
echo "Connected successfully";
?>

Inserting Rows into the Database

To insert data into the MySQL table, you will first need to create an HTML form that matches your table structure. The form’s action attribute should point to a PHP file called “insert.php”, which will handle the actual insertion of data into the table.

Here is the code for the ‘insert.php’ file:

<?php
$servername = "sql109.epizy.com";
$username = "epiz_xxx";
$password = "xxxxx";
$dbname = "epiz_xxx";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);
// Check connection
if ($conn->connect_error) {
  die("Connection failed: " . $conn->connect_error);
}

$firstName = $_POST['firstName']
$lastName = $_POST['lastName']
$email = $_POST['email']

$sql = "INSERT INTO students (firstname, lastname, email) VALUES ('" . $firstName . "', '" . $lastName . "', '" . $email . "')";

if ($conn->query($sql) === TRUE) {
  echo "New record created successfully";
} else {
  echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>
